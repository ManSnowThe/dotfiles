### Yay and AUR packages

**Yay installation:**
```
sudo pacman -S --needed git base-devel
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si
```

**AUR packages:**
```
yay -S rose-pine-gtk-theme-full
yay -S rose-pine-cursor

yay -S visual-studio-code-bin
yay -S yaak-bin
```
